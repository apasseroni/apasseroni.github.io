TileQuest

TileQuest is a tile-merging game akin to "Threes" or "2048", but infused with elements commonly found in RPGs. Instead of numbers, tiles contain characters, items, and locations, which interact in interesting ways when they merge or collide with each other. The goal is to take the familiar RPG gameplay loop of slaying monsters, looting items, completing quests, and leveling up heroes, and package it as a mobile-friendly puzzle/strategy game. The game is currently in development.

Asteroid Miner

Asteroid Miner is an arcade style game that I developed as part of a week-long game jam where the goal was to reimagine the classic Atari arcade game, "Asteroids" with some sort of twist. I chose to add a mining mechanic where the player has to deploy astronauts to mine resources from asteroids to increase their score.

Gallery

AutoBattleBots

I had an idea for an auto-battle game inspired by BattleBots which resulted in this prototype. In this game, players assemble robots from modular parts and watch them battle in a round-based elimination tournament. As an auto-battler, each robot is completely controlled by AI, with their behavior being influenced only by the parts they have equipped.

Gallery | Play

Threes Clone

This was an educational project that I made to go through the process of making a complete game. In order to remove any non-technical roadblocks, I chose to create a "clone" of the popular mobile game "Threes" which I've always admired for its clever and minimalist design.
